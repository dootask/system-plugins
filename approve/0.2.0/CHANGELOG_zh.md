### 审批中心重写版

- **全新底座**：审批中心从旧 Go/Flowable 引擎整体重写为自包含全栈插件（TanStack Start + React 19 + SQLite），发起、审批、列表、详情、流程与表单设计全部在插件内，不再依赖主程序数据库与页面。
- **自定义表单**：支持单行/多行文本、数字、金额、日期、日期时间（精确到分钟）、日期范围、下拉、多选、人员、部门、明细子表、附件等字段类型，表单按配置动态渲染与校验。
- **全新界面与导航**：发起申请按分类（假勤/行政/财务/人事/其他）以模板宫格呈现并支持搜索；桌面端左侧边栏（我收到的·我提交的·管理）+ 移动端底部导航，整套界面采用 shadcn UI。
- **更贴心的细节**：列表、详情、参与人与流程时间线均展示成员头像；审批意见与评论支持上传图片并在时间线内查看；日期时间为「日期 + 时间」两个独立控件，请假、加班起止时间精确到时分；从某个列表进入详情后返回会回到该列表。
- **列表统一表格化**：待处理/已处理/抄送我的/已提交与模板管理均采用表格 + 搜索/筛选 + 分页（含总数与每页数量），数据多时浏览更清晰。
- **内置模板开箱即用**：内置请假、加班、报销、差旅申请、评审五个模板，首次启动自动播种，可复制修改。报销支持按金额自动升级到更高层级审批。
- **机器人通知**：审批待办、抄送、结果由「审批助手」机器人以卡片消息推送（仅展示内容），审批人到审批中心处理。
- **数据备份**：管理员可在「审批管理 · 数据备份」一键备份、还原、下载（zip 包含审批数据库与上传附件）。上传附件改由插件本地存储，故能随库一并备份。撤回、拒绝、删除等重要操作均二次确认。
- **旧数据自动迁移与无缝续跑**：首次启动一次性将旧审批数据迁入新库，已结束单据保留为历史、进行中流程由新引擎接续推进，升级即用，无需人工干预。
