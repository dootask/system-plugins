# 看不到审批 / 安装失败怎么办

## 问题
打开 DooTask 后在「应用」里看不到「审批」入口，或在应用市场点「安装」后长时间没反应、提示失败。

## 常见原因
- 审批是独立插件（应用市场 app id：`approve`），主程序不内置；未装时入口不出现
- 普通用户没有应用市场的安装权限，必须系统管理员安装
- 首次拉取镜像较慢，看起来像「卡住」实际仍在下载
- 服务器无法访问应用市场镜像源（网络 / 防火墙 / DNS 问题）

## 解决
1. 确认自己角色：非管理员就联系管理员安装
2. 管理员打开应用市场，搜索「审批」
3. 点击安装后，通过安装日志查看实时进度（不要立即重试）
4. 日志卡在拉取镜像：检查服务器到镜像源的网络连通性
5. 安装完成但入口不出现：刷新浏览器或重新登录一次
6. 安装失败可在应用市场重试，仍失败请保留日志反馈给运维

## 看不到管理菜单
「模板管理 / 数据备份 / 数据统计」只有管理员能看到，普通用户看不到是正常的。
