# 在审批单上评论

## 是什么
评论让审批单的参与者随时补充说明、讨论、贴材料，而不影响审批流转。即使你不是当前审批人，只要能看到这张单子，就能评论。

## 操作步骤
1. 打开审批单详情
2. 在操作区文本框输入要说的话
3. 需要时点「+」上传图片
4. 点「仅评论」提交

## 说明
- 评论会出现在详情页时间线里，与审批记录混排，按时间倒序展示
- 支持随评论上传图片，图片以缩略图展示，点开可查看
- 评论不改变单子状态（不等于批准 / 拒绝 / 退回）

## 谁能评论
发起人、所有参与人（审批 / 抄送 / 加签）以及管理员，都能在能看到的审批单上评论。
