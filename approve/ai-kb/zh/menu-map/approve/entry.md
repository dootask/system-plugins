# 审批中心入口在哪

## 路径
- 桌面端：左侧栏「应用」→「审批」
- 移动端：底部 Tabbar「应用」→「审批」

## 页面结构
进入后左侧（移动端为底部 Tab / 分段控件）按用户视角分组：

**发起**
- **发起申请**：模板宫格，按分类浏览或搜索后选模板发起

**我收到的**
- **待处理**：分配给我、需要我批准 / 拒绝的审批，菜单旁带未处理数量角标
- **已处理**：我已经处理过的审批
- **抄送我**：流程把我设为抄送人的审批

**我发起的**
- **我发起**：我自己提交的所有审批单（含审批中 / 通过 / 拒绝 / 撤回各状态）

**管理（仅管理员可见）**
- **数据统计**：各状态审批单数量统计 + XLSX 导出
- **模板管理**：新建 / 编辑 / 启停 / 删除审批模板
- **数据备份**：手动备份、恢复、下载、删除备份

## 权限要求
- 任何登录用户都能看到「发起申请 / 待处理 / 已处理 / 抄送我 / 我发起」
- 「数据统计 / 模板管理 / 数据备份」仅管理员可见

## 看不到入口怎么办
1. 确认应用市场已安装 approve 插件，参见 [[approve.cannot-install.faq]]
2. 看不到管理菜单：当前用户不在管理员名单内
3. 安装后入口未出现：刷新浏览器或重新登录一次
