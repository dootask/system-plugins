# 审批中心是什么

## 定义
审批中心（approve）是 DooTask 的流程审批应用，用于在企业内提交、流转、处理表单类申请。典型场景：请假、加班、外出、报销、差旅、用印、采购、合同会签、评审。它是独立插件，安装后在「应用」中出现「审批」入口，所有操作都在该页面内完成。

## 三个核心对象
- **审批模板**：管理员预先设计的审批样板，包含「表单」和「审批流程」两部分。普通用户只能选用，不能修改，详见 [[approve.template.concept]]
- **审批单（实例）**：一次具体的申请；用户每发起一次就生成一张审批单，按模板定义的流程逐节点流转，详见 [[approve.instance-status.concept]]
- **流程节点**：模板里的步骤，常见类型有发起、审批、抄送、条件分支，详见 [[approve.node.concept]]

## 三类角色
- **发起人**：提交申请的人，可在无人处理前撤回，可全程补充评论
- **审批人**：在「待处理」看到任务，可批准 / 拒绝 / 退回，还能转交或加签他人
- **抄送人**：在「抄送我」看到知会，无需操作

## 主要能力
- 自定义表单（14 种字段类型）与可视化审批流程设计
- 按表单数值或单选项自动路由的条件分支
- 或签 / 会签 / 依次三种多人审批方式
- 转交、加签、退回重提等灵活流转
- 待处理 / 已处理 / 抄送我 / 我发起 四类列表 + 数据统计
- 管理端：模板管理、数据备份恢复、旧数据迁移、XLSX 导出
- 审批进度变化通过机器人在聊天里推送可点击卡片
