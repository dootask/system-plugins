# 审批插件元信息

## 定义
审批（approve）是 DooTask 的独立插件，由官方提供（应用市场 app id 为 `approve`）。主程序不内置审批逻辑，所有模板、表单、审批单、流转记录都由插件容器独立维护，通过反向代理把 `/apps/approve/` 路径挂载到 DooTask 界面。

## 关键属性
- **作者**：DooTask 官方
- **运行形态**：独立 Docker 容器（镜像 `dootask/approve`），作为 docker-compose 服务 `approve` 启动
- **数据存储**：插件自带数据库，连同上传的附件一起保存在插件的数据卷里；与主程序业务表完全隔离
- **身份打通**：使用主程序登录态识别用户，无需二次登录；用户 / 部门 / 角色等人员主数据来自主程序，审批不复制用户表
- **菜单注入**：安装后自动在「应用」注册「审批」入口
- **管理员**：由部署时配置的管理员名单决定谁能进管理端（模板管理 / 数据备份 / 数据统计），普通用户看不到这些菜单

## 安装与启用
1. 应用市场搜索「审批」
2. 点击安装，等待镜像拉取与容器启动
3. 安装完成后插件自动启用，「审批」入口立即出现
4. 首次启动会自动生成一批内置模板（请假、加班、报销、差旅、评审等），可直接发起

## 升级与卸载
- 升级：应用市场出现新版本时点击「更新」
- 卸载：应用市场对应插件 →「卸载」；卸载后入口消失

## 不支持
- 主程序无法直接 SQL 查询审批数据，必须经插件页面或接口
- 不能离线安装到不联网的环境（需访问应用市场镜像源）
