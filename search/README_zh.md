# Manticore 智能搜索插件

## 简介
Manticore Search 是一款高性能开源搜索引擎，专为 DooTask 系统提供智能文件内容搜索能力。它支持全文搜索、KNN 向量搜索和混合搜索，让您可以通过语义理解找到所需文件。

## 主要特点
- **混合搜索**：同时支持关键词匹配和语义相似度搜索
- **向量搜索**：内置 KNN 向量搜索，采用 HNSW 算法
- **内容搜索**：支持搜索文件内容，而不仅仅是文件名
- **高性能**：搜索响应极快，资源占用低
- **MySQL 兼容**：完全兼容 MySQL 协议，易于集成
- **中文支持**：内置中文分词（ICU）

## 适用场景
- 需要搜索文件内容（Word、Excel、PDF、文本等）
- 希望通过自然语言描述找到相关文件
- 需要更智能的搜索体验，不仅仅是关键词匹配

## 搜索类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 关键词搜索 | 传统全文匹配 | 搜索 "季度报告" |
| 语义搜索 | AI 理解搜索意图 | 搜索 "财务分析文档" 可找到 "Q3 收入报表" |
| 混合搜索 | 结合两者优势 | 更精准的搜索结果 |

## 注意事项
- 首次安装后需要等待系统完成文件内容索引
- 向量搜索需要配合 AI 助手插件使用（用于生成 Embedding）
- 建议系统至少有 2GB 可用内存
- 大文件（>1MB）的内容提取可能需要较长时间
- 相比其他搜索方案更加轻量级

## 技术规格
- 向量维度：1536（兼容 OpenAI text-embedding-3-small）
- 存储引擎：Manticore Search 15.x
- 索引类型：KNN 向量索引 (HNSW) + 倒排全文索引
- 中文分词：ICU Chinese 形态分析
